# Design Context (for imperfect-to-exact skills)

## Target audience
Бренды, агентства и создатели контента, которым нужен премиальный видеомонтаж для продаж/удержания: от коротких Reels до бренд-фильмов.

## Use cases
Одностраничный landing для лида: быстро показать стиль монтажа, типы проектов (соцсети/коммерция/YouTube/клипы) и подтолкнуть к контакту. Без CMS, только статические плейсхолдеры, чтобы легко заменить контент.

## Brand personality / tone
Cinematic luxury + modern agency. Тёмный премиум, кинематографичная композиция, глубина (blur/glow), аккуратная motion-оркестрация, “дорого” и самобытно, не шаблонно.

## Differentiator (одно запоминающееся)
Hero как film-grade кадр + cursor-following liquid background (ваш `components/ui/liquid.js`) и тонкие кинематографичные оверлеи/переходы между секциями.

## Constraints / technical requirements
- Next.js App Router + TypeScript + Tailwind
- Framer Motion для reveal/hero animations
- Lucide React для иконок
- Один-page layout с anchor navigation
- Полностью responsive и производительная верстка под Vercel
- Без добавления лишних библиотек / бэкенда / auth

## Visual constraints
По вашему ТЗ фон должен быть визуально “чёрным” (допустим near-black), чтобы `liquid.js` давал эффект по курсору без серых/корпоративных градиентов.
